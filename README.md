Migrating to this repo because this [repository](https://github.com/JAgbanwa/miscellaneousmathproblems/tree/main) is very crowded and messy with many files. This repo aims to store/contain 
attempts (solutions at best) to this catalogue of open polynomial diophantine problems from this [paper](https://arxiv.org/pdf/2404.08518).

This [folder](https://github.com/JAgbanwa/PolynomialDiophantineEquations/tree/main/diophantine_infinitely_many_solutions_polished) attempts to solve 
$y^2 + x^3 \cdot y + z^2 + 1 = 0$ from Table 13 of the aforementioned paper.
